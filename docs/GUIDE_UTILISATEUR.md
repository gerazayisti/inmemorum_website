# 🕊️ Guide Utilisateur : Mémorial Familial Sawa

Bienvenue dans votre espace d'administration. Ce guide vous explique comment gérer le mémorial de votre proche en toute simplicité.

## 🔒 Accès à l'Administration
1. Connectez-vous via le bouton "Admin" (en bas à droite sur mobile si vous êtes connecté).
2. Ou rendez-vous directement sur : `votre-site.com/admin/dashboard`.

---

## 🛠️ Les Sections de Gestion

### 1. Hommage (Portrait & Accueil)
Modifiez l'identité, les dates, ainsi que le message de bienvenue.
- **Le Nom** et les **Dates** s'affichent en grand sur l'accueil.
- **L'Introduction** permet de raconter brièvement qui était la personne.

### 2. Biographie (Ligne de Vie)
Ajoutez des moments marquants année par année.
- Cliquez sur **"Ajouter une étape"**.
- Renseignez l'année (ex: 1980), le titre et un descriptif.
- Ces étapes s'affichent dans une ligne du temps animée sur le site public.

### 3. Modération (Livre d'Or) 🛡️
C'est ici que vous gérez les messages laissés par les visiteurs.
- **Par défaut**, les messages ne sont pas visibles sur le site.
- Vous devez les **"Approuver"** ✅ pour qu'ils s'affichent.
- Vous pouvez aussi **"Supprimer"** 🗑️ les messages inappropriés.

### 4. Galerie (Souvenirs) 📸
Importez des photos ou des vidéos directement depuis votre téléphone ou ordinateur.
- Cliquez sur **"Ajouter un média"**.
- Sélectionnez votre fichier.
- Ajoutez une légende si vous le souhaitez pour expliquer le contexte de l'image.

### 5. Agenda (Commémorations)
Ne manquez plus les moments importants de recueillement.
- Ajoutez des dates (anniversaires, messes, etc.).
- Activez **"Événement annuel"** pour que la date revienne chaque année automatiquement.

---

## ⚙️ Paramètres & Confidentialité
Dans l'onglet **Paramètres**, vous pouvez :
- Modifier le **Titre du site** (qui apparaît dans le navigateur).
- **Masquer ou Afficher** des rubriques entières. Par exemple, si vous n'avez pas encore fini de remplir la Galerie, vous pouvez la masquer temporairement.

## 💡 Conseils Sawa Premium
- **Photos** : Privilégiez des photos de haute qualité pour le portrait d'accueil.
- **Textes** : L'utilisation du *Vouvoiement* ou de termes respectueux est recommandée pour conserver l'aspect solennel du mémorial.
